# Hilo
Hilo is a easy game. It means that you have two card aleatory. Once you see the first card you need to guess if the next card is higher o lower. Then the program show the score. If you acert you gain 100 points but if you not acert you lose 75 points. If you lose all your points the game is over.

## Getting Started
---
Make sure you have Python 3.8.0 or newer installed and running on your machine. Open a terminal and 
browse to the project's root folder. Start the program by running the following command.
```
python3 dice 
```
You can also run the program from an IDE like Visual Studio Code. Start your IDE and open the 
project folder. Select the main module inside the dice folder and click the "run" button.

## Project Structure
---
The project files and folders are organized as follows:
```
root                    (project root folder)
+-- Hilo                (source code for game)
  +-- game              (specific classes)
  +-- __main__.py       (program entry point)
+-- README.md           (general info)
```
